# user-manager
User manager using nodejs-express-pug

## Pasos a seguir:
-"Clone or download" el repo

-Desde la consola cd /path/to/repo

### -Con docker
```bash
docker build -t user-manager .
docker run -p 8080:8080 user-manager
```

### -Con npm
```bash
npm install
npm start
```

-En el navegador http://localhost:8080

Done!
